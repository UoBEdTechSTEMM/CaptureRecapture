Capture-Recapture Experiment

Procedure
1. Take the tank of fish below.
2. Select the size of the sub-sample of the fish. This is n1. Write it in the table below. This is the first capture event.
3. Mark these fish.
4. Put the fish back in the tank. The fish will swim about, so their positions become randomised.
5. Select the size of the second sub-sample of the fish. This is n2. Write it in the table below.
6. Count the number of marked fish that were recaptured. This is m2, put it in the table.
7. Put the fish back in the tank.
8. Repeat the recapture process three more times (n1 will obviously remain the same), varying the number you recapture.
9. Replace the fish in the tank.
10. Calculate the estiamted population size (N) for each of the three capture events. N is given by: $$ N = \frac{\mathbf{n}_1\mathbf{n}_2}{\mathbf{m}_2} $$
